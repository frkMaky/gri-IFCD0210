Cómo crear páginas web bien hechas

1. No te límites a herramientas visuales

Existen muchas herramientas incluso gratuitas, que te permiten crear páginas web sin saber realmente HTML.

Esto es una manera muy rápida de crear páginas web porque te "oculta" el código y trabajas de una manera similar a como ya estas acostumbrado a hacerlo en los editores de texto. Y realmente está muy bien para hacer, por ejemplo, una primera versión de una página web o parte de ella.

Pero no caigas en la tentación de dejar de aprender HTML a fondo por ello. Esto sería pan para hoy y hambre para mañana porque así siempre tendrás un techo muy bajo, cosas un poco más avanzadas no se pueden hacer así y, por tanto, siempre tendrás un techo muy bajo.

2. Piensa una buena estructura común

Antes de crear una página web, procura pensar una estructura lógica común compuesta por etiquetas de divisiones.

Un ejemplo puede ser, por ejemplo, una estructura básica de cabecera / cuerpo / pie .

Cuando quieres aplicar diseño CSS a la página resultada mucho más fácil y verás como una estructura bien pensada te dará mucho más juego para crear un buen diseño.
