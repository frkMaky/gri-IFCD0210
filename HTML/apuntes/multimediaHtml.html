Multimedia en HTML

Multimedia en la web es sonido, música, vídeos, películas y animaciones.

¿Qué es la multimedia?

Multimedia viene en muchos formatos diferentes. Puede ser casi cualquier cosa que puedas oír o ver, como imágenes, música, sonido, vídeos, discos, películas, animaciones y más.

Las páginas web suelen contener elementos multimedia de diferentes tipos y formatos.

Soporte del navegador

Los primeros navegadores web admitían únicamente texto, limitado a una única fuente en un solo color.

Más tarde llegaron los navegadores compatibles con colores, fuentes, imágenes y multimedia.

Formatos multimedia

Los elementos multimedia (como audio o vídeo) se almacenan en archivos multimedia.

La forma más común de descubrir el tipo de un archivo es mirar la extensión del archivo.

Los archivos multimedia tienen formatos y diferentes extensiones como: .wav, .mp3, .mp4, .mpg, .wmv y .avi.

Formatos de vídeo comunes

Existen muchos formatos de vídeo.

Los formatos MP4, WebM y Ogg son compatibles con HTML.

YouTube recomienda el formato MP4.

Format File Description
Ogg .ogg Theora Ogg. Developed by the Xiph.Org Foundation. Supported by HTML.
WebM .webm WebM. Developed by Mozilla, Opera, Adobe, and Google. Supported by HTML.
MPEG-4 o MP4 .mp4 MP4. Developed by the Moving Pictures Expert Group. Commonly used in video cameras and TV hardware. Supported by all browsers and recommended by YouTube.

Nota: El HTML estándar solo admite vídeos MP4, WebM y Ogg.

Formatos de audio comunes

MP3 es el mejor formato para música grabada comprimida. El término MP3 se ha convertido en sinónimo de música digital.

Si su sitio web trata sobre música grabada, MP3 es la opción.

Format File Description
AAC .aac AAC (Advanced Audio Coding). Developed by Apple as the default format for iTunes. Plays well on Apple computers, but not in web browsers.
WAV .wav WAV. Developed by IBM and Microsoft. Plays well on Windows, Macintosh, and Linux operating systems. Supported by HTML.
Ogg .ogg Ogg. Developed by the Xiph.Org Foundation. Supported by HTML.
MP3 .mp3 MP3 files are actually the sound part of MPEG files. MP3 is the most popular format for music players. Combines good compression (small files) with high quality. Supported by all browsers.

Nota: El estándar HTML solo admite audio MP3, WAV y Ogg.
