Entidades en HTML

Entidades de caracteres HTML

Algunos caracteres están reservados en HTML.

Si utiliza los signos menor que (<) o mayor que (>) en su texto HTML, el navegador puede mezclarlos con etiquetas.

Se pueden utilizar nombres de entidades o números de entidades para mostrar caracteres HTML reservados.

Los nombres de las entidades se ven así:

 &entity_name;

Los números de entidad se ven así:

&#entity_number;

Los nombres de las entidades son más fáciles de recordar que los números de las entidades.

Espacio irrompible

Una entidad HTML comúnmente utilizada es el espacio sin separación:

Un espacio que no se separa es un espacio que no se dividirá en una nueva línea.

Dos palabras separadas por un espacio que no se rompa se mantendrán juntas (no se dividirán en una nueva línea). Esto es útil cuando romper las palabras puede resultar perjudicial.

Ejemplos:

  • § 10
  • 10 kilómetros por hora
  • 10 p.m.

Otro uso común del espacio sin separación es evitar que los navegadores trunquen espacios en las páginas HTML.

Si escribe 10 espacios en su texto, el navegador eliminará 9 de ellos. Para agregar espacios reales a su texto, puede utilizar la función entidad de carácter.

El guión sin separación ( ‐ ) se utiliza para definir un carácter de guión (‐) que no se divide en una nueva línea.

Combinando

Algunas marcas diacríticas, como las graves ( ̀ ) y las agudas ( ́ ), se llaman acentos.

Las marcas diacríticas se pueden utilizar en combinación con caracteres alfanuméricos para producir un carácter que no está presente en el conjunto de caracteres (codificación) utilizado en la página.

Símbolos HTML

Los símbolos o letras que no están presentes en su teclado se pueden agregar a HTML mediante entidades.

Entidades de símbolos HTML

Las entidades HTML se describieron en el capítulo anterior.

Muchos símbolos matemáticos, técnicos y monetarios no están presentes en un teclado normal.

Para agregar dichos símbolos a una página HTML, puede usar el nombre de la entidad o el número de la entidad (una referencia decimal o hexadecimal) para el símbolo:

            Muestra el signo del euro:

            <p>I will display &euro;</p>
            <p>I will display &#8364;</p>
            <p>I will display &#x20AC;</p>
        
            Se mostrará como:

            Mostraré €
            Mostraré €
            Mostraré €

¿Qué son los emojis?

Los emojis parecen imágenes o íconos, pero no lo son.

Son letras (caracteres) del juego de caracteres UTF-8 (Unicode).

UTF-8 cubre casi todos los caracteres y símbolos del mundo.

El atributo del juego de caracteres HTML

Para mostrar una página HTML correctamente, un navegador web debe conocer el juego de caracteres utilizado en la página.

Esto se especifica en la etiqueta <meta>

            <meta charset="UTF-8"%gt;

Si no se especifica, UTF-8 es el juego de caracteres predeterminado en HTML.

Caracteres UTF-8

Muchos caracteres UTF-8 no se pueden escribir en un teclado, pero siempre se pueden mostrar usando números (llamados números de entidad):

  • A es 65
  • B es 66
  • C es 67
            <!DOCTYPE html>
            <html>
            <meta charset="UTF-8">
            <body>
            <p>I will display A B C</p>
            <p>I will display A B C</p>
            </body>
            </html>

Ejemplo explicado

El elemento <meta charset="UTF-8"> define el juego de caracteres.

Los caracteres A, B y C se muestran junto a los números 65, 66 y 67.

Para que el navegador comprenda que está mostrando un carácter, debe comenzar el número de entidad con y terminar con ; (punto y coma).

Personajes emojis

            <!DOCTYPE html>
            <html>
              <meta charset="UTF-8">

            <body>
                <h1>My First Emoji</h1>
                <p>😀</p>
            </body>
            </html>

Dado que los emojis son caracteres, se pueden copiar, mostrar y cambiar de tamaño como cualquier otro carácter en HTML.

            <!DOCTYPE html>
            <html>
            <meta charset="UTF-8">
            <body>
                <h1>Sized Emojis</h1>
                <p style="font-size:48px">
                    😀 😄 😍 💗
                </p>
            </body>
            </html>
