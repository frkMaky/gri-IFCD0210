Encabezados de tabla HTML

Las tablas HTML pueden tener encabezados para cada columna o fila, o para muchas columnas/filas.

Encabezados de tabla HTML

Los encabezados de las tablas se definen con elementos th. Cada elemento th representa una celda de la tabla.

            <table>
                <tr>
                    <th>Firstname</th>
                    <th>Lastname</th>
                    <th>Age</th>
                </tr>
                <tr>
                    <td>Jill</td>
                    <td>Smith</td>
                    <td>50</td>
                </tr>
                <tr>
                    <td>Eve</td>
                    <td>Jackson</td>
                    <td>94</td>
                </tr>
            </table>

Encabezados de mesa verticales

Para utilizar la primera columna como encabezados de tabla, defina la primera celda de cada fila como elemento <th>

            <table>
                <tr>
                    <th>Firstname</th>
                    <th>Lastname</th>
                    <th>Age</th>
                </tr>
                <tr>
                    <td>Jill</td>
                    <td>Smith</td>
                    <td>50</td>
                </tr>
                <tr>
                    <td>Eve</td>
                    <td>Jackson</td>
                    <td>94</td>
                </tr>
            </table>

Alinear encabezados de tabla

De forma predeterminada, los encabezados de las tablas están en negrita y centrados

Nombre Apellido Edad
Jill Herrero 50
Vispera Jackson 94

Para alinear a la izquierda los encabezados de la tabla, use la propiedad text-align CSS

            th {
                text-align: left;
            }

Encabezado para varias columnas

Puede tener un encabezado que abarque dos o más columnas.

Para hacer esto, use el atributo colspan en el elemento <th>

Nombre Edad
Jill Herrero 50
Vispera Jackson 94 
            <table>
            <tr>
                <th colspan="2">Nombre</th>
                <th>Edad</th>
            </tr>
            <tr>
                <td>Jill</td>
                <td>Vispera</td>
            </tr>
            <tr>
                <td>50</td>
                <td>94</td>
            </tr>
            </table>

Título de la tabla

Puede agregar un título que sirva como encabezado para toda la tabla.

Monthly savings
Month Savings
January $100
February $50

Para agregar un título a una tabla, use la etiqueta <caption>

Nota: La etiqueta <caption> debe insertarse inmediatamente después de la etiqueta <table>

            <table>
            <caption>Monthly savings</caption>
            <tr>
