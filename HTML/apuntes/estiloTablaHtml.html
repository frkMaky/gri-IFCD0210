Estilo de la tabla en HTML

Utilice CSS para que sus tablas se vean mejor.

Tabla HTML - Rayas de cebra

Si agrega un color de fondo en cada fila de la tabla, obtendrá un bonito efecto de rayas de cebra.

Para aplicar estilo a todos los demás elementos de fila de la tabla, use el selector:nth-child(even) de esta manera:

            tr:nth-child(even) {
                background-color: #D6EEEE;
            }

Nota: Si usa (odd)en lugar de (even), el estilo se producirá en las filas 1,3,5, etc. en lugar de 2,4,6, etc.

Tabla HTML: rayas verticales de cebra

Para hacer rayas de cebra verticales, diseñe cada dos columnas , en lugar de cada dos filas .

Configure los elementos :nth-child(even) como este:

            td:nth-child(even), th:nth-child(even) {
                background-color: #D6EEEE;
            }

Nota: Coloque el selector :nth-child()en los elementos th y td si desea tener el estilo tanto en los encabezados como en las celdas normales de la tabla.

Combina rayas de cebra verticales y horizontales

Puede combinar el estilo de los dos ejemplos anteriores y tendrá rayas en filas y columnas alternas.

Si utilizas un color transparente conseguirás un efecto de superposición.

Utilice un color rgba()para especificar la transparencia del color:

            tr:nth-child(even) {
                background-color: rgba(150, 212, 212, 0.4);
            }
            th:nth-child(even),td:nth-child(even) {
                background-color: rgba(150, 212, 212, 0.4);
            }

Divisores Horizontales

Especifica bordes solo en la parte inferior de cada fila de la tabla, tendrá una tabla con divisiones horizontales.

Agregue la propiedad border-bottom a todos los elementos tr para obtener divisiones horizontales:

            tr {
                border-bottom: 1px solid #ddd;
            }

Resaltar filas al pasar el mouse

Utilice el selector:hover sobre tr para resaltar las filas de la tabla al pasar el mouse:

            tr:hover {background-color: #D6EEEE;}
