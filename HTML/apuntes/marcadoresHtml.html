Enlaces HTML: crear marcadores

Los enlaces HTML se pueden utilizar para crear marcadores, de modo que los lectores puedan saltar a partes específicas de una página web.

Crear un marcador en HTML

Los marcadores pueden resultar útiles si una página web es muy larga.

Para crear un marcador, primero cree el marcador y luego agréguele un enlace.

Cuando se hace clic en el enlace, la página se desplazará hacia abajo o hacia arriba hasta la ubicación con el marcador.

Primero, use el atributo id para crear un marcador:

            <h2 id="C4">Capítulo 4</h2>

Luego, agregue un enlace al marcador ("Saltar al Capítulo 4"), desde la misma página:

            <a href="#C4">Saltar al Capítulo 4 </a>

También puedes agregar un enlace a un marcador en otra página:

            <a href="html_demo.html#C4">Jump to Chapter 4</a>

Resumen del capítulo

  • Utilice el atributo id (id="valor") para definir marcadores en una página
  • Utilice el atributo href (href="# valor") para vincular al marcador
