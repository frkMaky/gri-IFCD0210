Citas en HTML

En este capítulo veremos los elementos <blockquote>, <q>, <abbr>, <address>, <cite> y <bdo> HTML.

<blockquote> para citas

El elemento de HTML <blockquote> define una sección que se cita de otra fuente.

Los navegadores suelen aplicar sangría a los elementos <blockquote>.

            <p>Here is a quote from WWF's website:</p>
            <blockquote cite="http://www.worldwildlife.org/who/index.html">
            For 60 years, WWF has worked to help people and nature thrive. As the world's leading conservation
            organization, WWF works in nearly 100 countries. At every level, we collaborate with people around the
            world to develop and deliver innovative solutions that protect communities, wildlife, and the places in
            which they live.
            </blockquote>

HTML <q> para citas breves

La etiqueta de HTML <q> define una cita breve.

Los navegadores normalmente insertan comillas alrededor de la cita.

            <p>WWF's goal is to: <q>Build a future where people live in harmony with nature.</q></p>

<abbr> para abreviaturas

La etiqueta HTML <abbr> define una abreviatura o un acrónimo, como "HTML", "CSS", "Mr.", "Dr.", "ASAP", "ATM".

Marcar abreviaturas puede proporcionar información útil a los navegadores, sistemas de traducción y motores de búsqueda.

Consejo: utilice el atributo title para mostrar la descripción de la abreviatura/acrónimo cuando pase el mouse sobre el elemento.

            <p>The <abbr title="World Health Organization">WHO</abbr> was founded in 1948.</p>

<address> para información de contacto

La etiqueta de HTML <address> define la información de contacto del autor/propietario de un documento o artículo.

La información de contacto puede ser una dirección de correo electrónico, URL, dirección física, número de teléfono, identificador de redes sociales, etc.

El texto del elemento <address> generalmente se muestra en cursiva y los navegadores siempre agregarán un salto de línea antes y después del elemento <address>.

            <address>
                Written by John Doe.<br>
                Visit us at:<br>
                Example.com<br>
                Box 564, Disneyland<br>
                USA
            </address>

<cite> para el título de un trabajo

La etiqueta de HTML <cite> define el título de una obra creativa (por ejemplo, un libro, un poema, una canción, una película, un cuadro, una escultura, etc.).

NotaEl nombre de una persona no es el título de una obra.

El texto del elemento <cite> normalmente se muestra en cursiva.

            <p><cite>The Scream</cite> by Edvard Munch. Painted in 1893.</p>

<bdo> para sobreescritura bidireccional

BDO significa sobreescritura bidireccional.

La etiqueta HTML <bdo> se utiliza para cambiar la forma de escritura del texto actual:

            <bdo dir="rtl">This text will be written from right to left</bdo>

Ejercicio:

Utilice un elemento HTML para agregar comillas alrededor de las letras "Frío".

            <p>
            Soy tan Frío.
            </p>
