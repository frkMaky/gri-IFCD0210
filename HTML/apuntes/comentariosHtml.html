Comentarios en HTML

Los comentarios en HTML no se muestran en el navegador, pero pueden ayudar a documentar su código fuente HTML.

Etiqueta de comentario en HTML

Puede agregar comentarios a su código HTML utilizando la siguiente sintaxis:

<!-- Write your comments here -->

Observe que hay un signo de exclamación (!) en la etiqueta inicial, pero no en la etiqueta final.

Nota:El navegador no muestra los comentarios, pero pueden ayudar a documentar el código fuente HTML.

Añadir comentarios

Con comentarios puedes colocar notificaciones y recordatorios en tu código HTML:

            <!-- This is a comment -->
            <p>This is a paragraph.</p>
            <!-- Remember to add more information here -->

Ocultar contenido

Los comentarios se pueden utilizar para ocultar contenido.

Esto puede resultar útil si oculta contenido temporalmente:

            <p>This is a paragraph.</p>
            <!-- <p>This is another paragraph </p> -->
            <p>This is a paragraph too.</p>

También puedes ocultar más de una línea. Todo lo que esté entre <!-- y --> quedará oculto en la pantalla.

Ocultar una sección de código HTML:

            <p>This is a paragraph.</p>
            <!--
            <p>Look at this cool image:</p>
            <img border="0" src="pic_trulli.jpg" alt="Trulli">
            -->
            <p>This is a paragraph too.</p>

Los comentarios también son excelentes para depurar HTML, porque puedes comentar líneas de código HTML, una a la vez, para buscar errores.

Ocultar contenido en línea

Los comentarios se pueden utilizar para ocultar partes en medio del código HTML.

Ocultar una parte de un párrafo:

            <p>This <!-- great text -->is a paragraph.</p>

Ejercicio:

Utilice la etiqueta de comentario HTML para crear un comentario a partir del texto "Esto es un comentario".

            <h1>Esto es un encabezado</h1>
            esto es un comentario
            <p>Esto es un párrafo.</p>
