Listas ordenadas en HTML

La etiqueta de HTML <ol> define una lista ordenada. Una lista ordenada puede ser numérica o alfabética.

Lista HTML ordenada

Una lista ordenada comienza con la etiqueta <ol>. Cada elemento de la lista comienza con la etiqueta <li>.

Los elementos de la lista estarán marcados con números de forma predeterminada:

            <ol>
                <li>Coffee</li>
                <li>Tea</li>
                <li>Milk</li>
            </ol>

Lista HTML ordenada: el atributo de tipo

El atributo type de la etiqueta <ol> define el tipo de marcador de elemento de la lista:

tipo Descripción
type="1" Los items de la lista serán enumerados con números (por defecto)
type="A" Los items de la lista serán enumerados con letras mayúsculas
type="a" Los items de la lista serán enumerados con letras minúsculas
type="I" Los items de la lista serán enumerados con números romanos en mayúscula
type="i" Los items de la lista serán enumerados con números romanos en minúscula

Números:

            <ol type="1">
                <li>Coffee</li>
                <li>Tea</li>
                <li>Milk</li>
            </ol>

Letras mayúsculas:

            <ol type="A">
                <li>Coffee</li>
                <li>Tea</li>
                <li>Milk</li>
            </ol>

Letras minúsculas:

            <ol type="a">
                <li>Coffee</li>
                <li>Tea</li>
                <li>Milk</li>
            </ol>

Números romanos en mayúsculas:

            <ol type="I">
                <li>Coffee</li>
                <li>Tea</li>
                <li>Milk</li>
            </ol>

Números romanos en minúsculas:

            <ol type="i">
                <li>Coffee</li>
                <li>Tea</li>
                <li>Milk</li>
            </ol>

Inicio del conteo

De forma predeterminada, una lista ordenada comenzará a contar desde 1. Si desea comenzar a contar desde un número específico, puede usar el atributo start

            <ol start="50">
                <li>Coffee</li>
                <li>Tea</li>
                <li>Milk</li>
            </ol>

Listas HTML anidadas

Las listas se pueden anidar (una lista dentro de otra lista):

            <ol>
                <li>Coffee</li>
                <li>Tea
                <ol>
                    <li>Black Tea</li>
                    <li>Green Tea</li>
                </ol>
                </li>
                <li>Milk</li>
            </ol>

Nota: Un elemento de lista (<li>) puede contener una nueva lista y otros elementos HTML, como imágenes y enlaces, etc.

Resumen del capítulo

  • Utilice el elemento <ol> de HTML para definir una lista ordenada
  • Utilice el atributo type de HTML para definir el tipo de numeración
  • Utilice el elemento <li> de HTML para definir un elemento de lista
  • Las listas se pueden anidar
  • Los elementos de la lista pueden contener otros elementos HTML
