Párrafos HTML

El elemento de HTML <p> define un párrafo.

Un párrafo siempre comienza en una nueva línea y los navegadores agregan automáticamente un espacio en blanco (un margen) antes y después de un párrafo.

            <p>This is a paragraph.</p>
            <p>This is another paragraph.</p>

Cómo se muestra el HTML

No puede estar seguro de cómo se mostrará el HTML.

Las pantallas grandes o pequeñas y las ventanas redimensionadas crearán resultados diferentes.

Con HTML, no puede cambiar la visualización agregando espacios o líneas adicionales en su código HTML.

El navegador eliminará automáticamente los espacios y líneas adicionales cuando se muestre la página:

            <p>
            This paragraph
            contains a lot of lines
            in the source code,
            but the browser
            ignores it.
            </p>

            <p>
            This paragraph
            contains a lot       of spaces
            in the source code,
            but the browser
            ignores it.
            </p>

Rayas horizontales en HTML

La etiqueta <hr> define una ruptura temática en una página HTML y suele mostrarse como una raya horizontal.

El elemento <hr> se utiliza para separar contenido (o definir un cambio) en una página HTML:

            <1>This is heading 1</h1>
            <p>This is some text.</p>
            <hr>
            <h2>This is heading 2</h2>
            <p>This is some other text.</p>
            <hr>

La etiqueta <hr> es una etiqueta vacía, lo que significa que no tiene etiqueta de fin.

Saltos de línea HTML

El elemento de HTML <br> define un salto de línea.

Use <br> si desea un salto de línea (una nueva línea) sin comenzar un nuevo párrafo:

            <p>This is<br>a paragraph<br>with line breaks.</p>

La etiqueta <br> es una etiqueta vacía, lo que significa que no tiene etiqueta final.

El problema del poema

Este poema se mostrará en una sola línea:

            <p>
                My Bonnie lies over the ocean.
                My Bonnie lies over the sea.
                My Bonnie lies over the ocean.
                Oh, bring back my Bonnie to me.
            </p>

Solución: el elemento HTML <pre>

El elemento HTML <pre> define texto preformateado.

El texto dentro de un elemento <pre> se muestra en una fuente de ancho fijo (normalmente Courier) y conserva tanto los espacios como los saltos de línea:

            <pre>
                My Bonnie lies over the ocean.
                My Bonnie lies over the sea.
                My Bonnie lies over the ocean.
                Oh, bring back my Bonnie to me.
            </pre>

Referencia de etiquetas HTML

Etiqueta Descripción
<p> Define un párrafo
<hr> Define un cambio temático en el contenido
<br> Inserta un salto de línea
<pre> Define texto preformateado
