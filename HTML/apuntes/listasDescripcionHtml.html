Otras listas en HTML

HTML también admite listas de descripciones (o de definiciones).

Listas de descripción HTML

Una lista de descripción es una lista de términos, con una descripción de cada término.

La etiqueta <dl> define la lista de descripción, la etiqueta <dt> define el término (nombre) y la etiqueta <dd> describe cada término

            <dl>
                <dt>Coffee</dt>
                <dd>- black hot drink</dd>
                <dt>Milk</dt>
                <dd>- white cold drink</dd>
            </dl>

Resumen del capítulo

  • Utilice el elemento <dl> HTML para definir una lista de descripción
  • Utilice el elemento <dt> HTML para definir el término de descripción.
  • Utilice el elemento <dd> HTML para describir el término en una lista de descripción

Ejercicio

Utilice los elementos HTML correctos para hacer una descripción de cada término en la lista de descripción.

            <dl>
                <dt>Café</dt>
                - bebida caliente negra
                <dt>Leche</dt>
                - bebida fría blanca
            </dl>

Etiquetas de lista en HTML

Etiqueta Descripción
<dl> Define una lista de descripción
<dt> Define un término en una lista de descripción
<dd> Describe el término en una lista de descripción
