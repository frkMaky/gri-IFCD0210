TEST selectores de atributos:

  1. Selecciona aquellas imágenes con extensión png: 
                    img[src$=".png"]
  2. Se muestran de color azul todos los enlaces que tengan un atributo «class», independientemente de su valor: 
                    a[class] {
                    color:blue;
                }
  3. Se muestran de color azul todos los enlaces que tengan un atributo «class» con el valor «externo»: 
                    a[class="externo"]{
                    color:blue;
                }
  4. Se muestran de color azul todos los enlaces que apunten al sitio «http://www.ejemplo.com». 
                    a[href="http://www.ejemplo.com"] {
                    color:blue;
                }
  5. Selecciona todos los elementos de la página cuyo atributo «lang» sea igual a «en», es decir, todos los elementos en inglés 
                    *[lang="en"]
  6. Selecciona todos los elementos de la página cuyo atributo «lang» empiece por «es», es decir, «es», «es-ES», «es-AR», etc. 
                    *[lang|="es"]
  7. Selecciona todos los enlaces que no tengan el atributo href. 
                    a:not(a[href])
  8. Utilizando selectores de atributos realiza los siguientes ejercicios:
    • Selecciona aquellas imágenes con extensión png y añade un borde de 5 píxeles de color rojo. 
                              img[src$=".png"]{
                            border: 5px solid red;
                        }
    • Muestra de color verde todos los enlaces que tengan un atributo “class”, independientemente de su valor. 
                              a[class] {
                            color:green;
                        }
    • Muestra de color rosa todos los enlaces que apunten al sitio “https://www.eniun.com”. 
                              a[href="https://www.eniun.com"]{
                            color:pink;
                        }
    • Muestra de color lila todos los enlaces que tengan un atributo “class” con el valor “externo”. 
                              a[class="externo"] {
                            color:violet;
                        }
  9. Oculta todos los elementos de la página cuyo atributo "lang" sea igual a "en", es decir, todos los elementos en inglés:
    • *[lang="en"] { display:none; }
    • ^[lang=en] { display:none; }
    • ^[lang=en]$ { display:none; }
