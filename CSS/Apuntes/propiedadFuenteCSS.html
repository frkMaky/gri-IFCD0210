Propiedad de fuente en CSS

La propiedad de fuente CSS

Para acortar el código, también es posible especificar todas las propiedades de fuente individuales en una sola propiedad.

La propiedad font es una propiedad abreviada para:

  • font-style
  • font-variant
  • font-weight
  • font-size/line-height
  • font-family

Nota: Los valores font-size y font-family son obligatorios. Si falta uno de los otros valores, se utiliza su valor predeterminado.

Ejemplo

Use font para establecer varias propiedades de fuente en una declaración:

            p.a {
                font: 20px Arial, sans-serif;
            }
            p.b {
                font: italic small-caps bold 12px/30px Georgia, serif;
            }

Ejercicio:

Establezca la fuente para <h1> en "Verdana".

                <style>
                    h1 {
                        : Verdana;
                    }
                </style>
                <body>
                    <h1>Este es un encabezado</h1>
                    <p>Este es un párrafo</p>
                </body>

Todas las propiedades de fuentes CSS

Propiedad Descripción
font Establece todas las propiedades fuente en una declaración
font-family Especifica tipo de fuente para el texto
font-size Especifica el tamaño del fuente del texto
font-style Especifica el estilo de fuente para el texto
font-variant Especifica si o no un texto debería ser mostrado en letras mayúsculas pequeñas
font-weight Especifica el peso de una fuente
