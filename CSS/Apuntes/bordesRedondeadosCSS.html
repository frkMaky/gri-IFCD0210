Bordes redondeados en CSS

La propiedad border-radius se utiliza para agregar bordes redondeados a un elemento:

            p {
                border: 2px solid red;
                border-radius: 5px;
            }

Ejercicio

Todas las propiedades del borde en CSS

Propiedad Descripción
border Establece todas las propiedades border en una declaración
border-bottom Establece todas las propiedades border-bottom en una declaración
border-bottom-color Establece el color del borde de abajo
border-bottom-style Establece el estilo del borde de abajo
border-bottom-width Establece el ancho del borde de abajo
border-color Establece el color de los 4 bordes
border-left Establece todas las propiedades border-left en una declaración
border-left-color Establece el color del borde izquierdo
border-left-style Establece el estilo del borde izquierdo
border-left-width Establece el ancho del borde izquierdo
border-radius Establece los 4 bordes con esquinas redondeadas
border-right Establece todas las propiedades border-right en una declaración
border-right-color Establece el color del borde derecho
border-right-style Establece el estilo del borde derecho
border-right-width Establece el ancho del borde derecho
border-style Establece el estilo para los 4 bordes
border-top Establece todas las propiedades border-top en una declaración
border-top-color Establece el color del borde de arriba
border-top-style Establece el estilo del borde de arriba
border-top-width Establece el ancho del borde de arriba
border-width Establece el ancho de los 4 bordes
