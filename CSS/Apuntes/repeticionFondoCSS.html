Repetición de la imagen de fondo en CSS

Repetición del fondo CSS

De forma predeterminada, la propiedad background-image repite una imagen tanto horizontal como verticalmente.

Si se repite solo horizontalmente ( background-repeat: repeat-x;):

Ejemplo

            body {
                background-image: url("gradient_bg.png");
                background-repeat: repeat-x;
            }

Consejo: Para repetir una imagen verticalmente, establezca background-repeat: repeat-y;

Fondo sin repetición

La propiedad background-repeat:no-repeat también especifica mostrar la imagen de fondo solo una vez :

Ejemplo

Muestra la imagen de fondo solo una vez:

            body {
                background-image: url("img_tree.png");
                background-repeat: no-repeat;
            }

En el ejemplo anterior, la imagen de fondo se coloca en el mismo lugar que el texto. Queremos cambiar la posición de la imagen, para que no perturbe demasiado el texto.

Background-position de CSS

La propiedad background-position se utiliza para especificar la posición de la imagen de fondo.

Ejemplo

Coloque la imagen de fondo en la esquina superior derecha:

            body {
                background-image: url("img_tree.png");
                background-repeat: no-repeat;
                background-position: right top;
            }

Las propiedades de posición y repetición de fondo de CSS

Propiedad Descripción
background-position Establece la posición de inicio de una imagen de fondo
background-repeat Establece cómo se establece una imagen de fondo
