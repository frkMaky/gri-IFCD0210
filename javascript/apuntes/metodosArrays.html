Métodos de array de JavaScript

Longitud del array

La propiedad length devuelve la longitud (tamaño) de un array:

            const fruits = ["Banana", "Orange", "Apple", "Mango"];
            let size = fruits.length;

toString()

El método de JavaScript toString()convierte un array en una cadena de valores del array (separados por comas).

            const fruits = ["Banana", "Orange", "Apple", "Mango"];
            document.getElementById("demo").innerHTML = fruits.toString();

Resultado: Banana,Orange,Apple,Mango

at()

ES2022 introdujo el método at():

            Obtenga el tercer elemento de frutas usando at():
            const fruits = ["Banana", "Orange", "Apple", "Mango"];
            let fruit = fruits.at(2);

            Obtén el tercer elemento de las frutas usando []:
            const fruits = ["Banana", "Orange", "Apple", "Mango"];
            let fruit = fruits[2];

El método at() devuelve un elemento indexado de un array.

El método at() devuelve lo mismo que [].

El método at() es compatible con todos los navegadores modernos desde marzo de 2022:

Chrome 92 Edge 92 Firefox 90 Safari 15.4 Opera 78
Apr 2021 Jul 2021 Jul 2021 Mar 2022 Aug 2021

Nota Muchos idiomas permiten negative bracket indexing como [-1] acceder a elementos desde el final de un objeto/array/cadena. Esto no es posible en JavaScript porque [] se usa para acceder tanto a arrays como a objetos. obj[-1] se refiere al valor de la clave -1, no a la última propiedad del objeto.

El método at() se introdujo en ES2022 para resolver este problema.

join()

El método join() también une todos los elementos de el array en una cadena.

Se comporta igual que toString(), pero además puedes especificar el separador:

            const fruits = ["Banana", "Orange", "Apple", "Mango"];
            document.getElementById("demo").innerHTML = fruits.join(" * ");

Resultado: Banana * Orange * Apple * Mango

pop() y push()

Cuando trabaja con arrays, es fácil eliminar elementos y agregar nuevos elementos.

Sacar elementos de un array o insertar elementos en un array.

pop()

El método pop() elimina el último elemento de un array:

            const fruits = ["Banana", "Orange", "Apple", "Mango"];
            fruits.pop();

El método pop() devuelve el valor que "sacó":

            const fruits = ["Banana", "Orange", "Apple", "Mango"];
            let fruit = fruits.pop();

push()

El método push() agrega un nuevo elemento a un array (al final):

            const fruits = ["Banana", "Orange", "Apple", "Mango"];
            fruits.push("Kiwi");

El método push() devuelve la nueva longitud del array:

            const fruits = ["Banana", "Orange", "Apple", "Mango"];
            let length = fruits.push("Kiwi");

Elementos desplazamiento

shift() es equivalente a pop, pero trabajando en el primer elemento en lugar del último.

shift()

El método shift() elimina el primer elemento del array y "desplaza" todos los demás elementos a un índice inferior.

            const fruits = ["Banana", "Orange", "Apple", "Mango"];
            fruits.shift();

El método shift() devuelve el valor que fue "desplazado":

            const fruits = ["Banana", "Orange", "Apple", "Mango"];
            let fruit = fruits.shift();

unshift()

El método unshift() agrega un nuevo elemento a un array (al principio) y "desplaza" los elementos más antiguos:

            const fruits = ["Banana", "Orange", "Apple", "Mango"];
            fruits.unshift("Lemon");

El método unshift() devuelve la nueva longitud del array:

            const fruits = ["Banana", "Orange", "Apple", "Mango"];
            fruits.unshift("Lemon");

Elementos cambiantes

Se accede a los elementos del array utilizando su número de índice :

Los índices del array comienzan con 0:

[0] es el primer elemento del array
[1] es el segundo
[2] es el tercero...

            const fruits = ["Banana", "Orange", "Apple", "Mango"];
            fruits[0] = "Kiwi";

length

La propiedad length proporciona una manera sencilla de agregar un nuevo elemento a un array:

            const fruits = ["Banana", "Orange", "Apple", "Mango"];
            fruits[fruits.length] = "Kiwi";

delete()

Advertencia ! El uso de delete() deja agujeros undefined en el array.

Utilice pop() o shift() en su lugar.

            const fruits = ["Banana", "Orange", "Apple", "Mango"];
            delete fruits[0];

Fusionar arrays (concatenar)

En los lenguajes de programación, la concatenación significa unir cadenas de un extremo a otro.

La concatenación "bola de " y "nieve" da "bola de nieve".

Concatenar arrays significa unir arrays de un extremo a otro.

Concatenación de array de JavaScript()

El método concat() crea un nuevo array fusionando (concatenando) arrays existentes:

            const myGirls = ["Cecilie", "Lone"];
            const myBoys = ["Emil", "Tobias", "Linus"];

            const myChildren = myGirls.concat(myBoys);

Nota El método concat() no cambia los arrays existentes. Siempre devuelve un nuevo array.

El método concat() puede tomar cualquier número de argumentos del array.

            Ejemplo (fusión de tres arrays)
            const arr1 = ["Cecilie", "Lone"];
            const arr2 = ["Emil", "Tobias", "Linus"];
            const arr3 = ["Robin", "Morgan"];

            const myChildren = arr1.concat(arr2, arr3);

El método concat() también puede tomar cadenas como argumentos:

            Ejemplo (Fusionar un array con valores)
            const arr1 = ["Emil", "Tobias", "Linus"];
            const myChildren = arr1.concat("Peter");

copyWithin()

El método copyWithin() copia elementos del array a otra posición en un array:

            Copie al índice 2, todos los elementos del índice 0:
            const fruits = ["Banana", "Orange", "Apple", "Mango"];
            fruits.copyWithin(2, 0);
        
            Copiar al índice 2, los elementos del índice 0 al 2:
            const fruits = ["Banana", "Orange", "Apple", "Mango", "Kiwi"];
            fruits.copyWithin(2, 0, 2);

Nota El método copyWithin() sobrescribe los valores existentes.

El método copyWithin() no agrega elementos al array.

El método copyWithin() no cambia la longitud del array.

flat()

Aplanar un array es el proceso de reducir la dimensionalidad de un array.

Es útil cuando desea convertir un array multidimensional en un array unidimensional.

ES2019 Se introdujo el método flat().

El método flat() crea un nuevo array con elementos de subarray concatenados a una profundidad especificada.

            const myArr = [[1,2],[3,4],[5,6]];
            const newArr = myArr.flat();

splice() y slice() en arrays

El método splice() agrega nuevos elementos a un array.

El método slice()corta una parte de un array.

splice()

El método splice() se puede utilizar para agregar nuevos elementos a un array:

            const fruits = ["Banana", "Orange", "Apple", "Mango"];
            fruits.splice(2, 0, "Lemon", "Kiwi");

El primer parámetro (2) define la posición donde se deben agregar nuevos elementos.

El segundo parámetro (0) define cuántos elementos se deben eliminar .

El resto de parámetros ("Limón", "Kiwi") definen los nuevos elementos a añadir .

El método splice() devuelve un array con los elementos eliminados:

            const fruits = ["Banana", "Orange", "Apple", "Mango"];
            fruits.splice(2, 2, "Lemon", "Kiwi");

Usando splice() para eliminar elementos

Con una configuración de parámetros inteligente, puede utilizar splice() para eliminar elementos sin dejar "agujeros" en el array:

            const fruits = ["Banana", "Orange", "Apple", "Mango"];
            fruits.splice(0, 1);

El primer parámetro (0) define la posición donde se deben agregar nuevos elementos.

El segundo parámetro (1) define cuántos elementos se deben eliminar .

El resto de parámetros se omiten. No se agregarán nuevos elementos.

toSpliced()

ES2023 agregó el método toSpliced() como una forma segura de empalmar un array sin alterar el array original.

La diferencia entre el nuevo método toSpliced() y el antiguo método splice() es que el nuevo método crea un nuevo array, manteniendo el array original sin cambios, mientras que el método anterior alteró el array original.

            const months = ["Jan", "Feb", "Mar", "Apr"];
            const spliced = months.toSpliced(0, 1);

slice()

El método slice() divide una parte de un array en un nuevo array:

            Corte una parte de un array a partir del elemento 1 de el array ("Naranja"):
            const fruits = ["Banana", "Orange", "Lemon", "Apple", "Mango"];
            const citrus = fruits.slice(1);

Nota El método slice() crea una nueva array.

El método slice() no elimina ningún elemento de el array fuente.

            Corte una parte de un array a partir del elemento 3 de el array ("Apple"):

            const fruits = ["Banana", "Orange", "Lemon", "Apple", "Mango"];
            const citrus = fruits.slice(3);

El método slice() puede tomar dos argumentos como slice(1, 3).

Luego, el método selecciona elementos desde el argumento inicial y hasta (pero sin incluir) el argumento final.

            const fruits = ["Banana", "Orange", "Lemon", "Apple", "Mango"];
            const citrus = fruits.slice(1, 3);

Si se omite el argumento final, como en los primeros ejemplos, el método slice() corta el resto del array.

            const fruits = ["Banana", "Orange", "Lemon", "Apple", "Mango"];
            const citrus = fruits.slice(2);

toString()

JavaScript convierte automáticamente un array en una cadena separada por comas cuando se espera un valor primitivo.

Este es siempre el caso cuando intenta generar un array.

Estos dos ejemplos producirán el mismo resultado:

            const fruits = ["Banana", "Orange", "Apple", "Mango"];
            document.getElementById("demo").innerHTML = fruits.toString();
        
            const fruits = ["Banana", "Orange", "Apple", "Mango"];
            document.getElementById("demo").innerHTML = fruits;

NotaTodos los objetos JavaScript tienen un método toString().
