Iteraciones en arrays

foreach()

El método forEach() llama a una función una vez para cada elemento del array.

            const numbers = [45, 4, 9, 16, 25];
            let txt = "";
            numbers.forEach(myFunction);
    
            function myFunction(value, index, array) {
                txt += value + "<br>";
            }
Ejemplo

Tenga en cuenta que la función toma 3 argumentos:

  • El valor del item
  • El índice del item
  • El array en sí

El ejemplo anterior utiliza sólo el parámetro de valor. El ejemplo se puede reescribir como:

            const numbers = [45, 4, 9, 16, 25];
            let txt = "";
            numbers.forEach(myFunction);
            
            function myFunction(value) {
                txt += value + "<br>";
            }

map()

El método map() crea un nuevo array realizando una función en cada elemento del array.

El método map() no ejecuta la función para elementos de array sin valores.

El método map() no cambia el array original.

Este ejemplo multiplica cada valor de array por 2:

            const numbers1 = [45, 4, 9, 16, 25];
            const numbers2 = numbers1.map(myFunction);

            function myFunction(value, index, array) {
                return value * 2;
            }

Tenga en cuenta que la función toma 3 argumentos:

  • El valor del item
  • El índice de items
  • el array en sí

Cuando una función de devolución de llamada usa solo el parámetro de valor, se pueden omitir los parámetros de índice y array:

            const numbers1 = [45, 4, 9, 16, 25];
            const numbers2 = numbers1.map(myFunction);

            function myFunction(value) {
                return value * 2;
            }
Ejemplo

flatMap()

ES2019 agregó el método Array flatMap() a JavaScript.

El método flatMap() primero asigna todos los elementos de un array y luego crea un nuevo array.

            const myArr = [1, 2, 3, 4, 5, 6];
            const newArr = myArr.flatMap((x) => x * 2);

filter()

El método filter() crea un nuevo array con elementos del array que pasan una prueba.

Este ejemplo crea un nuevo array a partir de elementos con un valor mayor que 18:

            const numbers = [45, 4, 9, 16, 25];
            const over18 = numbers.filter(myFunction);

            function myFunction(value, index, array) {
                return value > 18;
            }

Tenga en cuenta que la función toma 3 argumentos:

  • El valor del item
  • El índice del item
  • el array en sí

En el ejemplo anterior, la función de devolución de llamada no utiliza los parámetros de índice y array, por lo que se pueden omitir:

            const numbers = [45, 4, 9, 16, 25];
            const over18 = numbers.filter(myFunction);

            function myFunction(value) {
                return value > 18;
            }

reduce()

El método reduce() ejecuta una función en cada elemento del array para producir (reducirlo a) un valor único.

El método reduce() funciona de izquierda a derecha en el array.

El método reduce() no reduce el array original.

Este ejemplo encuentra la suma de todos los números en un array:

            const numbers = [45, 4, 9, 16, 25];
            let sum = numbers.reduce(myFunction);

            function myFunction(total, value, index, array) {
                return total + value;
            }

Tenga en cuenta que la función toma 4 argumentos:

  • El total (el valor inicial/valor devuelto previamente)
  • El valor del item
  • El índice del item
  • el array en sí

El ejemplo anterior no utiliza los parámetros de índice y array. Se puede reescribir como:

            const numbers = [45, 4, 9, 16, 25];
            let sum = numbers.reduce(myFunction);

            function myFunction(total, value) {
                return total + value;
            }

El método reduce() puede aceptar un valor inicial:

            const numbers = [45, 4, 9, 16, 25];
            let sum = numbers.reduce(myFunction, 100);

            function myFunction(total, value) {
                return total + value;
            }

El método reduceRight() ejecuta una función en cada elemento del array para producir (reducirlo a) un valor único.

reduceRight() funciona de derecha a izquierda en el array.

El método reduceRight() no reduce el array original.

Este ejemplo encuentra la suma de todos los números en un array:

            const numbers = [45, 4, 9, 16, 25];
            let sum = numbers.reduceRight(myFunction);

            function myFunction(total, value, index, array) {
                return total + value;
            }

Tenga en cuenta que la función toma 4 argumentos:

  • El total (el valor inicial/valor devuelto previamente)
  • El valor del item
  • El índice del item
  • el array en sí

El ejemplo anterior no utiliza los parámetros de índice y array. Se puede reescribir como:

            const numbers = [45, 4, 9, 16, 25];
            let sum = numbers.reduceRight(myFunction);

            function myFunction(total, value) {
                return total + value;
            }

every()

El método every() comprueba si todos los valores del array pasan una prueba.

Este ejemplo comprueba si todos los valores del array son mayores que 18:

            const numbers = [45, 4, 9, 16, 25];
            let allOver18 = numbers.every(myFunction);

            function myFunction(value, index, array) {
                return value > 18;
            }

Tenga en cuenta que la función toma 3 argumentos:

  • El valor del item
  • El índice del item
  • el array en sí

Cuando una función de devolución de llamada usa solo el primer parámetro (valor), los demás parámetros se pueden omitir:

            const numbers = [45, 4, 9, 16, 25];
            let allOver18 = numbers.every(myFunction);

            function myFunction(value) {
                return value > 18;
            }

some()

El método some() comprueba si algunos valores del array pasan una prueba.

Este ejemplo comprueba si algunos valores del array son mayores que 18:

            function myFunction(value, index, array) {
                return value > 18;
            }

Tenga en cuenta que la función toma 3 argumentos:

  • El valor del item
  • El índice de item
  • el array en sí

Array.from()

El método Array.from() devuelve un objeto Array de cualquier objeto con una propiedad de longitud o cualquier objeto iterable.

Ejemplo

Cree una array a partir de una cadena:

            Array.from("ABCDEFG");

Soporte del navegador

keys()

El método Array.keys()devuelve un objeto Array Iterator con las claves de un array.

Ejemplo

Cree un objeto Array Iterator, que contenga las claves del array:

            const fruits = ["Banana", "Orange", "Apple", "Mango"];
            const keys = fruits.keys();
            
            for (let x of keys) {
                text += x + "<br>";
            }

entries()

Cree un Array Iterator y luego repita los pares clave/valor:

            const fruits = ["Plátano", "Naranja", "Manzana", "Mango"];
            const f = fruits.entries();
            
            for (let x of f) {
                document.getElementById("demo").innerHTML += x;
            }

El método entries() devuelve un objeto Array Iterator con pares clave/valor:

            [0, "Plátano"]
            [1, "Naranja"]
            [2, "Manzana"]
            [3, "Mango"]

El método entries() no cambia el array original.

with()

ES2023 agregó el método with() como una forma segura de actualizar elementos en un array sin alterar el array original.

            const months = ["Januar", "Februar", "Mar", "April"];
            const myMonths = months.with(2, "March");

Extensión de array de JavaScript (...)

El operador ... expande un iterable (como un array) en más elementos:

            const q1 = ["Jan", "Feb", "Mar"];
            const q2 = ["Apr", "May", "Jun"];
            const q3 = ["Jul", "Aug", "Sep"];
            const q4 = ["Oct", "Nov", "May"];
            const year = [...q1, ...q2, ...q3, ...q4];
