Búsquedas en arrays

indexOf()

El método indexOf() busca en un array el valor de un elemento y devuelve su posición.

Nota: El primer elemento tiene la posición 0, el segundo elemento tiene la posición 1, y así sucesivamente.

            Busque en un array el elemento "Apple":

            const fruits = ["Apple", "Orange", "Apple", "Mango"];
            let position = fruits.indexOf("Apple") + 1;

Sintaxis

array.indexOf(item,comienzo)
item Requerido. El item a buscar
comienzo Opcional. En donde comienza la búsqueda. Valores negativos comenzarán en la posición dada contando desde en final, y buscando al final

Array.indexOf()devuelve -1 si no se encuentra el item.

Si el elemento está presente más de una vez, devuelve la posición de la primera aparición.

lastIndexOf()

Array.lastIndexOf() es igual que Array.indexOf(), pero devuelve la posición de la última aparición del elemento especificado.

            Busque en un array el elemento "Apple":

            const fruits = ["Apple", "Orange", "Apple", "Mango"];
            let position = fruits.lastIndexOf("Apple") + 1;

Sintaxis

array.indexOf(item,comienzo)
item Requerido. El item a buscar
comienzo Opcional. Donde comienza la búsqueda. Valores negativos comenzarán en la posición dada de conteo desde el final, y buscarán al principio

includes()

ECMAScript 2016 introdujo Array.includes()a los arrays. Esto nos permite verificar si un elemento está presente en un array (incluido NaN, a diferencia de indexOf).

            const fruits = ["Banana", "Orange", "Apple", "Mango"];
            fruits.includes("Mango"); // is true

Sintaxis

array.includes(item-de-búsqueda)

Array.includes() permite comprobar los valores de NaN. A diferencia de Array.indexOf().

Soporte del navegador

includes() es una característica de ECMAScript 2016 .

find()

El método find() devuelve el valor del primer elemento del array que pasa una función de prueba.

Este ejemplo encuentra (devuelve el valor de) el primer elemento mayor que 18:

            const numbers = [4, 9, 16, 25, 29];
            let first = numbers.find(myFunction);
            
            function myFunction(value, index, array) {
                return value > 18;
            }

Tenga en cuenta que la función toma 3 argumentos:

  • El valor del item
  • El índice del item
  • el array en sí

findLast()

ES2023 agregó el método findLast() que comenzará desde el final de un array y devolverá el valor del primer elemento que satisface una condición.

            const temp = [27, 28, 30, 40, 42, 35, 30];
            let high = temp.findLast(x => x > 40);
