Cadenas de literales (Template String) de JavaScript

Sintaxis de Back-Tics

Las Template Strings utilizan comillas invertidas (``) en lugar de comillas ("") para definir una cadena:

            let text = `Hello World!`;

Comillas dentro de cadenas

Las Template Strings permiten comillas simples y dobles dentro de una cadena:

            let text = `He's often called "Johnny"`;

Cadenas multilínea

Las Template Strings permiten cadenas multilínea:

            let text =
                `The quick
                brown fox
                jumps over
                the lazy dog`;

Interpolación

Template String proporciona una manera sencilla de interpolar variables y expresiones en cadenas.

El método se llama interpolación de cadenas.

La sintaxis es:

${...}

Sustituciones de variables

Las Template Strings permiten variables en cadenas:

            let firstName = "John";
            let lastName = "Doe";

            let text = `Welcome ${firstName}, ${lastName}!`;

La sustitución automática de variables por valores reales se denomina interpolación de cadenas .

Sustitución de expresiones

Las Template Strings permiten expresiones en cadenas:

            let price = 10;
            let VAT = 0.25;

            let total = `Total: ${(price * (1 + VAT)).toFixed(2)}`;

La sustitución automática de expresiones por valores reales se denomina interpolación de cadenas .

Template Strings en HTML

            let header = "Template Strings";
            let tags = ["template strings", "javascript", "es6"];

            let html = `<h2>${header}</h2><ul>`;
            for (const x of tags) {
                html += `<li>${x}</li>`;
            }
    
            html += `</ul>`;

Soporte del navegador

Template Strings es una característica de ES6 (JavaScript 2015).

ES6 es totalmente compatible con todos los navegadores modernos desde junio de 2017
