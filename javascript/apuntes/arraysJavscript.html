Arrays de JavaScript

Un array es una variable especial que puede contener más de un valor:

        const cars = ["Saab", "Volvo", "BMW"];

¿Por qué utilizar arrays?

Si tiene una lista de elementos (una lista de nombres de automóviles, por ejemplo), almacenar los automóviles en variables individuales podría verse así:

            let car1 = "Saab";
            let car2 = "Volvo";
            let car3 = "BMW";

Sin embargo, ¿qué pasa si quieres recorrer los coches y encontrar uno específico? ¿Y si no tuvieras 3 coches, sino 300?

¡La solución es un array!

Un array puede contener muchos valores bajo un solo nombre y puede acceder a los valores haciendo referencia a un número de índice.

Creando un array

Usar un literal de array es la forma más sencilla de crear un array de JavaScript.

            const array_name = [item1, item2, ...];

Es una práctica común declarar arrays con la palabra clave const .

            const cars = ["Saab", "Volvo", "BMW"];

Los espacios y los saltos de línea no son importantes. Una declaración puede abarcar varias líneas:

            const cars = [
                "Saab",
                "Volvo",
                "BMW"
            ];

También puedes crear un array y luego proporcionar los elementos:

            const cars = [];
            cars[0]= "Saab";
            cars[1]= "Volvo";
            cars[2]= "BMW";

Usando la palabra clave JavaScript new

El siguiente ejemplo también crea un array y le asigna valores:

            const cars = new Array("Saab", "Volvo", "BMW");

Los dos ejemplos anteriores hacen exactamente lo mismo.

No es necesario utilizar new Array().

Por simplicidad, legibilidad y velocidad de ejecución, utilice el método literal de array.

Accediendo a elementos del array

Se accede a un elemento del array haciendo referencia al número de índice :

            const cars = ["Saab", "Volvo", "BMW"];
            let car = cars[0];

Nota: Los índices del array comienzan con 0.

[0] es el primer elemento. [1] es el segundo elemento.

Cambiar un elemento del array

Esta declaración cambia el valor del primer elemento en cars:

            cars[0] = "Opel";
        
            const cars = ["Saab", "Volvo", "BMW"];
            cars[0] = "Opel";

Convertir un array en una cadena

El método de JavaScript toString()convierte un array en una cadena de valores de array (separados por comas).

            const fruits = ["Banana", "Orange", "Apple", "Mango"];
            document.getElementById("demo").innerHTML = fruits.toString();
            
            Resultado:
            Banana,Orange,Apple,Mango

Acceda al array completo

Con JavaScript, se puede acceder al array completo haciendo referencia al nombre del array:

            const cars = ["Saab", "Volvo", "BMW"];
            document.getElementById("demo").innerHTML = cars;

Los arrays son objetos

Los arrays son un tipo especial de objetos. El operador typeof en JavaScript devuelve "objeto" para arrays.

Pero los arrays de JavaScript se describen mejor como arrays.

Los arrays utilizan números para acceder a sus "elementos". En este ejemplo, person[0] devuelve John:

            Formación:
            const person = ["John", "Doe", 46];

Los objetos usan nombres para acceder a sus "miembros". En este ejemplo, person.firstName devuelve John:

            Objeto:
            const person = {firstName:"John", lastName:"Doe", age:46};

Los elementos del array pueden ser objetos

Las variables de JavaScript pueden ser objetos. Los arrays son tipos especiales de objetos.

Debido a esto, puedes tener variables de diferentes tipos en un mismo array.

Puedes tener objetos en un array. Puedes tener funciones en un array. Puedes tener arrays en un array:

            myArray[0] = Date.now;
            myArray[1] = myFunction;
            myArray[2] = myCars;

Propiedades y métodos del array

La verdadera fortaleza de los arrays de JavaScript son las propiedades y métodos integrados del array:

            cars.length // Returns the number of elements
            cars.sort() // Sorts the array

La propiedad length

La propiedad length de un array devuelve la longitud de un array (el número de elementos del array).

            const fruits = ["Banana", "Orange", "Apple", "Mango"];
            let length = fruits.length;

La propiedad length es siempre uno más que el índice del array más alto.

Accediendo al primer elemento del array

            const fruits = ["Banana", "Orange", "Apple", "Mango"];
            let fruit = fruits[0];

Accediendo al último elemento del array

            const fruits = ["Banana", "Orange", "Apple", "Mango"];
            let fruit = fruits[fruits.length - 1];

bucle para recorrer los elementos de un array

Una forma de recorrer un array es mediante un bucle for:

            const fruits = ["Banana", "Orange", "Apple", "Mango"];
            let fLen = fruits.length;
            let text = "<ul>";
            for (let i = 0; i < fLen; i++) {
                text += "<li>" + fruits[i] + "</li>";
            }
            text += "</ul>";

También puedes utilizar la función Array.forEach():

            const fruits = ["Banana", "Orange", "Apple", "Mango"];
            let text = "<ul>";
            fruits.forEach(myFunction);
            text += "</ul>";
            function myFunction(value) {
            text += "<li>" + value + "</li>";
            }

Agregar elementos del array

La forma más sencilla de agregar un nuevo elemento al array es mediante el método push():

            const fruits = ["Banana", "Orange", "Apple"];
            fruits.push("Lemon"); // Adds a new element (Lemon) to fruits

También se puede agregar un nuevo elemento a un array usando la propiedad length:

            const fruits = ["Banana", "Orange", "Apple"];
            fruits[fruits.length] = "Lemon"; // Adds "Lemon" to fruits

ADVERTENCIA ! Agregar elementos con índices altos puede crear "agujeros" no definidos en un array:

            const fruits = ["Banana", "Orange", "Apple"];
            fruits[6] = "Lemon"; // Creates undefined "holes" in fruits

Arrays asociativos

Muchos lenguajes de programación admiten arrays con índices de nombres.

JavaScript no admite arrays con índices de nombres.

En JavaScript, los arrays siempre usan índices numerados .

            const person = [];
            person[0] = "John";
            person[1] = "Doe";
            person[2] = 46;
            person.length; // Will return 3
            person[0]; // Will return "John"

ADVERTENCIA !! Si utiliza índices de nombres, JavaScript redefinirá el array como un objeto.

Después de eso, algunos métodos y propiedades del array producirán resultados incorrectos .

            const person = [];
            person["firstName"] = "John";
            person["lastName"] = "Doe";
            person["age"] = 46;
            person.length; // Will return 0
            person[0]; // Will return undefined

La diferencia entre arrays y objetos

En JavaScript, los arrays utilizan índices numerados .

En JavaScript, los objetos utilizan índices con nombre .

Los arrays son un tipo especial de objetos, con índices numerados.

Cuándo utilizar arrays. Cuándo utilizar objetos.

  • JavaScript no admite arrays asociativos.
  • Debe utilizar objetos cuando desea que los nombres de elementos sean cadenas (texto) .
  • Debe utilizar arrays cuando desee que los nombres de los elementos sean números .

new Array()

JavaScript tiene un constructor de arrays incorporado new Array().

Pero puedes usarlo de forma segura []en su lugar.

Estas dos declaraciones diferentes crean un nuevo array vacío llamado points:

            const points = new Array();
            const points = [];

Estas dos declaraciones diferentes crean un nuevo array que contiene 6 números:

            const points = new Array(40, 100, 1, 5, 25, 10);
            const points = [40, 100, 1, 5, 25, 10];

La palabra clave new puede producir algunos resultados inesperados:

            // Create an array with three elements:
            const points = new Array(40, 100, 1);
            // Create an array with two elements:
            const points = new Array(40, 100);
            // Create an array with one element ???
            const points = new Array(40);

Un error común

            const points = [40];

no es lo mismo que:

            const points = new Array(40);
            // Create an array with one element:
            const points = [40];
            // Create an array with 40 undefined elements:
            const points = new Array(40);

Cómo reconocer un array

Una pregunta común es: ¿Cómo sé si una variable es un array?

El problema es que el operador de JavaScript typeof devuelve " object":

            const fruits = ["Banana", "Orange", "Apple"];
            let type = typeof fruits;

El operador typeof devuelve un objeto porque un array de JavaScript es un objeto.

Solución 1:

Para resolver este problema, ECMAScript 5 (JavaScript 2009) definió un nuevo método Array.isArray():

            Array.isArray(fruits);

Solución 2:

El operador instanceof devuelve verdadero si un objeto es creado por un constructor determinado:

            const fruits = ["Banana", "Orange", "Apple"];
            fruits instanceof Array;
